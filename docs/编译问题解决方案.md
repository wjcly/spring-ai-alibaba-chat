# 编译错误解决方案

## 错误 1: Java 版本问题

**错误信息**: `无效的标记：--release` 或 `不支持的语言级别`

**原因**: 项目需要 JDK 17，但当前使用的是 JDK 8

### 解决方案

#### 方式一：安装 JDK 17（推荐）

**Windows 安装步骤**:

1. 下载 JDK 17:
   - Oracle JDK: https://www.oracle.com/java/technologies/downloads/#java17
   - OpenJDK: https://jdk.java.net/17/
   - Adoptium: https://adoptium.net/temurin/releases/?version=17

2. 安装后配置环境变量:
   ```
   JAVA_HOME=C:\Program Files\Java\jdk-17
   Path=%JAVA_HOME%\bin
   ```

3. 验证安装:
   ```cmd
   java -version
   ```
   应该显示 `java version "17.x.x"`

#### 方式二：使用 Maven 工具链（不升级 JDK）

如果你不能升级 JDK，可以修改项目使用 Java 8：

1. 修改 `pom.xml`:
   ```xml
   <properties>
       <java.version>8</java.version>
       <maven.compiler.source>8</maven.compiler.source>
       <maven.compiler.target>8</maven.compiler.target>
   </properties>
   ```

2. 但注意：Spring Boot 3.x 需要 Java 17+，你需要降级到 Spring Boot 2.7.x

## 错误 2: 缺少依赖包

**错误信息**: 
- `程序包 org.aspectj.lang 不存在`
- `程序包 org.springframework.data.jpa.repository 不存在`

**解决方案**: 已在 pom.xml 中添加以下依赖：

```xml
<!-- Spring Boot Data JPA -->
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-data-jpa</artifactId>
</dependency>

<!-- Spring Boot AOP (for AspectJ) -->
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-aop</artifactId>
</dependency>
```

## 错误 3: 导入包错误

**错误信息**: `找不到符号 IndustrialKnowledgeService`

**解决方案**: 已修复导入路径：
```java
// 错误
import com.example.ai.service.impl.IndustrialKnowledgeService;

// 正确
import com.example.ai.service.IndustrialKnowledgeService;
```

## 快速修复步骤

### Windows PowerShell (一键执行)

```powershell
# 1. 安装 JDK 17 (使用 Chocolatey)
choco install jdk17 --version=17.0.9

# 或者使用 winget
winget install Oracle.JavaRuntimeEnvironment

# 2. 重新编译项目
cd D:\备份\spring-ai-alibaba-demo
mvn clean compile -DskipTests
```

### 手动安装 JDK 17

1. 访问 https://adoptium.net/temurin/releases/?version=17
2. 下载 Windows x64 JDK
3. 运行安装程序
4. 配置环境变量
5. 重新编译

## 验证编译

```bash
cd D:\备份\spring-ai-alibaba-demo
mvn clean compile -DskipTests
```

成功输出应该显示：
```
[INFO] BUILD SUCCESS
[INFO] Total time:  XX.XXX s
```

## 如果仍然有问题

1. 清理 Maven 缓存：
   ```bash
   mvn clean
   ```

2. 删除本地仓库重新下载：
   ```bash
   rm -rf ~/.m2/repository/org/springframework
   rm -rf ~/.m2/repository/com/alibaba/cloud/ai
   ```

3. 使用详细输出编译：
   ```bash
   mvn compile -X
   ```
